== Description ==

Allows you to change administration menu titles and capabilities required to show menu item. Useful for hiding features in a CMS set up.

== Installation ==

1. Upload the `admin-menu-management` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set the options in the Settings Panel
